# Ecommerce Data Platform

A data infrastructure project for a multi-platform e-commerce business.

## Objective

Build a centralized PostgreSQL-based data platform to replace fragmented spreadsheet reporting across multiple sales channels.

## Current Focus

- define data sources
- design database schema
- standardize cross-platform sales reporting
- prepare for dashboarding and future automation

## Project Structure

- `docs/` project documentation
- `sql/` database schema
- `etl/` data ingestion scripts
- `data/` raw source files
- `dashboard/` reporting assets

## Planned Stack

- PostgreSQL
- Python
- ETL pipelines
- BI dashboards
- future browser automation / OpenClaw integration
